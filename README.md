# slitherlink-python
Jeu Slitherlink en Python avec fltk (librairie graphique de l'Université Gustave Eiffel basée sur tkinter)

L'objectif de ce projet de fin de première année était de reproduire le [Slitherlink](https://fr.puzzle-loop.com/) en y intégrant un "solveur" capable de résoudre automatiquement le casse-tête. Voir le [rapport](rapport.pdf) pour plus d'informations.

## Essayer sur [Replit](https://replit.com/@bruno00o/slitherlink?v=1)

https://replit.com/@bruno00o/slitherlink?v=1
